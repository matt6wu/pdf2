# PDF Reader - Git Submodule

这是一个Git子模块项目，指向原始仓库：

**原始仓库**: https://github.com/matt6wu/pdf2.git

## 项目说明
- **用途**: 现代化PDF阅读器 + 智能TTS朗读
- **技术**: HTML5 + PDF.js + JavaScript
- **端口**: 6788
- **PM2进程**: pdf-reader
- **域名**: https://pdf.mattwu.cc

## 核心功能
- PDF阅读、缩放、导航
- 智能TTS朗读（中英文）
- 响应式设计
- 拖拽上传

git clone https://github.com/matt6wu/pdf2.git
